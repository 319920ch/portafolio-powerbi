Proyectos

1. Seguimiento de Horas de Trabajo
Descripción: Dashboard operativo para el monitoreo de horas trabajadas por empleado y por área, con análisis de patrones de demanda según los horarios en que los clientes solicitan servicios.
Qué muestra:

Horas trabajadas por empleado y por período
Clientes y franjas horarias de mayor demanda
Comparativa entre carga real y capacidad esperada

Herramientas: Power BI, Power Query, DAX
Valor para el negocio: Permite a managers identificar picos de demanda y distribuir recursos con criterio, reemplazando seguimiento manual en hojas sueltas.

2. Gestión de Contratos
Descripción: Dashboard para el control de contratos vigentes con sus horas de trabajo asociadas, facilitando el seguimiento del estado contractual y la carga operativa vinculada a cada acuerdo.
Qué muestra:

Contratos activos, próximos a vencer y finalizados
Horas comprometidas vs. horas ejecutadas por contrato
Distribución de carga por cliente o proveedor

Herramientas: Power BI, Power Query, DAX
Valor para el negocio: Centraliza información contractual dispersa y permite anticipar vencimientos y desvíos de horas antes de que se conviertan en un problema.

3. Monitoreo de Base de Datos
Descripción: Dashboard técnico para el monitoreo continuo de la salud de la base de datos, con foco en uso de tablespaces, espacio de disco y estado de los backups RMAN.
Qué muestra:

Uso actual y tendencia de crecimiento por tablespace
Espacio de disco en el servidor
Estado y resultado de los backups RMAN (exitosos, fallidos, pendientes)

Herramientas: Power BI, Power Query, SQL
Valor para el negocio: Reemplaza la revisión manual de logs y permite detectar problemas de capacidad o fallos de backup antes de que afecten la operación.

Stack utilizado
Power BI · Power Query · DAX · Excel · SQL · PostgreSQL · SQL Server
